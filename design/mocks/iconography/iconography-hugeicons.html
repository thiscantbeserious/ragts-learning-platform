Size Classes Reference

Icon sizes are mapped to spacing tokens. Each size class applies a fixed width and height. All mask-image SVGs share a 20x20 viewBox for consistent scaling.

.icon--lg 24px --space-6 Upload zone, empty state
.icon--xl 32px --space-8 Hero, feature icons

Typography Baseline Match

Icons sized to sit inline with text at each type scale step. The icon height matches the computed line height of its paired text, keeping vertical rhythm.
