Draft G — Three Original Designs

Three fundamentally different approaches to representing 5-50+ navigable sections in a fixed-width vertical aside. Each design uses a different spatial metaphor: proportional trace map, hexagonal honeycomb, and dot matrix.

G1: Circuit Trace

A proportional minimap where each section is a horizontal trace whose height reflects its line count. The session's "shape" is immediately visible — heavy sections are tall traces, light sections are thin lines. A vertical backbone connects all traces like a circuit board bus. Active section glows with a chevron indicator. Tooltips appear to the left on hover.

G2: Hex Grid

Sections as hexagonal cells in a honeycomb layout. The staggered rows create a dense, organic grid that fits many items in a narrow column. Marker sections are brightly filled hexagons; detected sections use a subdued fill. Active hex pulses with a glow. Numbers inside each hex provide ordinal reference. Tooltips appear to the left.

G3: Dot Matrix

A dense grid of small square cells, evoking an LED status display or circuit board test points. Maximum information density in minimum space. Marker sections glow cyan; detected sections are dim neutral squares. Larger sections get slightly larger dots, creating a subtle texture that reveals the session's density pattern. Active section has a pulsing ring outline. Hover shows a connector-line tooltip.
