Iconography

Complete icon catalog for RAGTS. Stroke-based SVGs that inherit currentColor. Five size classes from 12px (xs) to 32px (xl). 52 icons across 6 groups.

Size Classes Reference

Icon sizes are mapped to spacing tokens. Each size class applies a fixed width and height. All SVGs share a 20x20 viewBox for consistent scaling.

Class Size Token Usage
.icon--xs 12px --space-3 Inline with small text
.icon--sm 16px --space-4 Badges, small buttons, inline
.icon--md 20px --space-5 Default, most components
.icon--lg 24px --space-6 Upload zone, empty state
.icon--xl 32px --space-8 Hero, feature icons

Typography Baseline Match

Icons sized to sit inline with text at each type scale step. The icon height matches the computed line height of its paired text, keeping vertical rhythm.
