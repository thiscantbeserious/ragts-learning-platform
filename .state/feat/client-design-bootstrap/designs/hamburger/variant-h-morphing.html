Implementation Notes

Complexity: LOW. Three absolutely-positioned spans with CSS transforms and transitions. No JS for the morph -- state classes toggle via Vue.

Three morph styles: (A) Compress then X -- classic, reliable. (B) Arrow morph -- top/bottom angle into a pointing chevron. (C) Pulse/breathe -- outer bars widen, middle shrinks, then X on active. All CSS-only.

Accessibility: prefers-reduced-motion media query disables all transforms. Button remains functional, just static. ARIA labels switch between "Menu" and "Close menu".
