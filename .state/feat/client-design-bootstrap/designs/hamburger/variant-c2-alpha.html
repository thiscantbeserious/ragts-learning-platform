Implementation Notes

Complexity: LOW. Pure CSS — one pseudo-element for the spine, three spans for bars. Letter identity is encoded in 3 custom properties (--bar-top, --bar-mid, --bar-bottom) which can be set at the component level or via a whitelabel config.

CSS-only feasibility: YES. No SVG, no JavaScript needed for rendering. Hover animation uses box-shadow transition. Bar width transitions work via CSS transition: width if animated letter-switching is ever desired.

Whitelabel API: Override --accent + bar width vars. The entire visual identity shifts with just 4 CSS custom properties. Color theming uses the same --accent-* family as BrandMark for consistency.
