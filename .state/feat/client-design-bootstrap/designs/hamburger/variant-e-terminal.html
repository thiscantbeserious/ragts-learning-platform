Variant E: Bracket / Terminal

E1 uses the universal terminal prompt ">_" — instantly recognizable to the target audience (developers browsing agent sessions). The underscore blinks on hover, adding life. E2 stacks three bracket pairs "[ ]" with interior dots, evoking collapsed code blocks or minimized terminal buffers. Both are typographic — rendered in the mono font — making them feel native to the terminal aesthetic without any brand-specific letter.
