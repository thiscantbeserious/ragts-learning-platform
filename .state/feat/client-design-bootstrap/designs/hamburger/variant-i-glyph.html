Implementation Notes

All pure CSS. No SVG, no icon fonts, no Unicode characters. Each glyph is built from positioned spans with backgrounds and border-radius. Guaranteed cross-platform consistency.

I-1 Grid Window -- reads as "dashboard panels". Very modern (Windows 11, macOS Launchpad). I-2 Stacked Planes -- cascade of content layers, close to hamburger but more spatial. I-3 Circuit Node -- TRON aesthetic, reads as "junction/hub". I-4 Terminal Prompt -- domain-specific: a terminal cursor with chevron bracket.

Risk: Non-standard icons may confuse first-time users. Mitigated by tooltip + aria-label. Grid Window (I-1) has strongest "nav" recognition.
