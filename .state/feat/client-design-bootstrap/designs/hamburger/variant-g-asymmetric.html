Implementation Notes

Complexity: VERY LOW. Same 3-bar structure as a standard hamburger, just with different width values per bar. No pseudo-elements needed.

Hover behavior: Bars animate to equal width (15px) on hover, creating a satisfying "snap to order" effect. Transition is 200ms ease-out.

Whitelabel: Bar patterns can be branded per-customer alongside accent color. Four presets shown (para, cascade, ascend, invert) but any 3 width values work.
