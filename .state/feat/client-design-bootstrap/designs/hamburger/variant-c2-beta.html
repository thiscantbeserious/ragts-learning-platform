Implementation Notes

Complexity: MEDIUM. Each bar is split into two segments with a gap between them. The notch position is controlled by segment widths per letter configuration. Requires 6 values per letter config (2 segment widths x 3 bars) but most bars are full-width (no notch).

CSS-only feasibility: YES. No SVG needed. Each bar row is a flex container with two span segments and a gap span. Notch width and segment widths are CSS custom properties. Could also use clip-path for a single-element approach, but flex segments are simpler.

Whitelabel API: Configure --accent color + notch-bar (top/mid/bottom) + notch-position (left/center/right). More expressive than Alpha but more config values. The notch is subtle enough that at 28px it reads primarily as hamburger.
