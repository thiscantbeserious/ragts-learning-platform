Variant B: Dot Grid / Minimal

B1 uses 3 vertical dots (3px, 4px gap) for a clean kebab-style icon. B2 uses a 2x3 dot grid (2.5px dots) for a more techy/sci-fi feel. Both are more minimal than bars and pair well with the TRON-neon aesthetic. Tradeoff: dots are less universally recognized as "menu" than bars.
