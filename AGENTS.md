# RAGTS Learning Platform - Agent Instructions

A self-hostable, white-label web platform for browsing, exploring, and learning from AI agent terminal sessions. Powered by [AGR](https://github.com/thiscantbeserious/agent-session-recorder) as the recording and transformation engine.

Read `README.md` for the project vision and `ARCHITECTURE.md` for system design.
